# Using Xbox w/ LFG Tool

Xbox players are able to invite others through Discord voice channels. To use with LFG Tool use the server's **"Create a VC"** channel.

---

# Step 1 - Link Your Xbox Account

To receive invites properly, your Xbox account must be connected to Discord.

1. Open Discord (mobile or desktop)  
2. Go to **User Settings**  
3. Click **Connections**  
4. Select **Xbox**  
5. Sign in and link your account  


---

# Step 2 - Create or Join a Voice Channel

1. Open Discord  
2. Go to the server where you're playing  
3. Find and join the **"Create a VC"** channel
4. Invite or wait for players to join  

We recommend also naming the voice channel to include "Xbox" in the name so players know which lobbies are for xbox.
You also don't have to speak in order to invite other players, as players join the voice call you should be able to invite them.

---

# Step 3 - Ensure someone is hosting & join their lobby.

Once you're in the same VC:

1. Ask the host to start the game lobby  
2. The host sends an invite via Xbox or in-game system  
3. You will receive the invite via:
   - Xbox notifications  
   - Discord activity panel (if supported)

4. Click **Join Game**
