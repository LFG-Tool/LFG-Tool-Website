If the LFG and VC interface and/or LFG lobbies are showing up as empty or blank this will show you how to fix it.

## How to fix it:

Here is a written guide, however you can use the image under this:
1. Go to your **user settings**, this in the bottom left of your discord.
2. Use the search bar and search for "embed"
3. Click "Appearance"
4. Enable **Show embeds and link previews**

> If this didn't work, you might have accidentally blocked the bot!

![Solution](/img/support/blanksolution1.png)
