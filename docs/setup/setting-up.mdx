# How to Setup the bot

### Inviting the Bot

1. [Invite the bot to your server](https://lfgtool.xyz/add)
2. We have a list of required permissions that the bot needs to function, if you're unsure please [read here](./permissions) on the specifics

### Commands

Run the `/configure` command.
    - The moderator role is so they can't get locked out of user created VCs.
    - The logging channels is for the VC logging.
    - VC Category: This is where TempVCs are placed.
    - Creator VC is the VC that people will join, the bot can create one or you can set it to an existing channel
    - SteamAppId is not required but recommended!
    - Join configuration is what people will use by default unless the branch has overwrites.

Please run the `/branch` command and use the "Create" button to create a branch.
    - A branch is what the user selects once they click the create LFG button, this is very important
    - The bot will also refuse to send the interface if you don't have at least 1 branch.
    - Examples include `Vanilla`, `Beta`, `Modded`, `Consoles`, `Steam` etc (Consoles and Steam could be used if they dont have cross platform play)

After you have all the branches, run `/interface` and select the channel.
    - You can always add branches later on without needing to resend this.
    - If you do not want VC or LFG services, feel free to delete the Embed you don't want.
